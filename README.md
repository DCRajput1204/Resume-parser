This project is an intelligent Resume Parser developed using Python and spaCy NLP to automatically extract structured information from unstructured resume documents.

The parser supports PDF, DOC, and DOCX formats and processes resumes to extract:

👤 Name

📧 Email Address

📱 Mobile Number

🛠 Skills (custom skills file supported)

🎓 Education Details

💼 Work Experience

📊 Total Years of Experience

🧠 Competencies

📈 Measurable Results

📄 Number of Pages

📝 Full Extracted Text

KEY FEATURES:

-Uses spaCy NLP model (en_core_web_sm) for entity recognition and text processing

-Custom skill matching using a predefined skills file

-Extracts structured data from unstructured resumes

-Handles multiple file formats (PDF, DOC, DOCX)

-Calculates total experience based on extracted experience data

-Supports multiprocessing for parsing multiple resumes efficiently

USE CASE:

This project can be integrated into:

-ATS (Applicant Tracking Systems)

-HR automation tools

-Recruitment platforms

-Resume screening systems
